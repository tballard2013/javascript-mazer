Javascript Mazer

  • A simple HTML, CSS, Javascript top-down maze game.
  • Show basic HTML, CSS to render a simple maze with an entrance and an exit.
  • User objective is to get through the maze.
  • User score influenced by time spent and number of moves between entrance and exit.
  • Javascript will be used to listen for user input and provide animation loop showing the game reacting to that input.
